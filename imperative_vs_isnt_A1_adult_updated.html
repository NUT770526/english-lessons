Imperative or Present Simple?

is / isn’t чи don’t?

Головне правило

is / isn’t
Використовуємо, коли говоримо, який хтось або щось є.
The test is easy.
Тест легкий.
The file isn’t empty.
Файл не порожній.
She isn’t busy.
Вона не зайнята.
Don’t + дія!
Використовуємо, коли говоримо людині: «Не роби це!»
Don’t buy it!
Не купуй це!
Don’t type it!
Не друкуй це!
Don’t sell the fish!
Не продавай рибу!
Не кажемо: Isn’t buy it!
Правильно: Don’t buy it!
