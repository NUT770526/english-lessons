Questions Practice

Goal: practise question words and correct word order in questions.
Read slowly. First choose the question word, then write the full question.

Before you start: DO / DOES / DID

Very important rule for questions:

Present Simple

now / every day / usually / often

I / you / we / they → DO

he / she / it → DOES

Question word + do/does + person + verb?

✅ Where do you work?

✅ What does Mike do?

Past Simple

yesterday / last week / last year / today

I / you / he / she / we / they → DID

Question word + did + person + verb?

✅ When did she get a promotion?

✅ Why did he quit?

Important: after DID use the base verb.

❌ Did she got? → ✅ Did she get?

⭐ Easy memory rule: Present = DO / DOES, but Past = DID.

Exercise 1. Choose the question word and make a question

Question words: where, what kind, what, how many

Example: I usually go to the cinema at the weekends. → When do you usually go to the cinema?

Exercise 2. Correct the mistakes

Write the correct question. Pay attention to word order: auxiliary verb + subject + main verb.
